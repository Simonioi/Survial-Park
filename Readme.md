This game was created as a Doom 1 homage, it's Survival First Person Shooter, in a forest with monsters and mutant plant dinosaur.

staying alive is an hardcore challenge.

monsters come in waves, with 3 at the start and +2 every wave. starting at wave 3 you have a stronger ennemy, and you have an additional one every three waves.

(the game get really hard starting at wave 9, with 19 monster and 3 dinosaurs.)
current best score :
Simon : Wave 11

# game modes :
the game comes in two mode : "Kill Room" and "Maze"

## Kill Room :
the kill room is a single room, with obstacle where the player is locked and has to survive the succeeding waves. currently the most playable mode, but the hardest one.

## Maze :
the maze is supposed to recreate a forest ,where the player is stranded and has to survive the monsters coming to eat him. the maze is proceduraly generated and has a few generation bugs.
### Disclaimer :
this game mode is a secondary mode, because the textures and 2.5D engine make the game really hard to play. please do not test if you have motion sickness, balance issues, or visual issues.

# controls :
Z : foreward
S : backward
Q : left
D : right
spacebar/mouseclick : shoot
escape : unlock the mouse and click on icons.

# Dev mode :
we added a dev mode option, with the 3D view, and a top 2D view. this mode is used to test and check the spawn and map generation. it as a teleport on click option, a spawn button, and clear all entities button.
